### 参考文献：

```
我要完成顶刊综述的部分写作任务，末尾会给你附上大纲，要求你根据大纲，给我梳理价值高的参考文献，参考文献占比：5年以上 10%，近五年 90%，近 3 年70%。直接给我文章链接，并给出二级标题下的写作思路。
```

### 写作：

```
按照你给出的参考文献和写作思路，帮我直接用中文完成写作，参考文献\cite{文章原题目}直接插入到对应位置，要求AI率 5% 以下，逻辑要严谨，语句要通顺，要求口语化占比 30%，学术用于占 70%。
```

### 翻译：

```
我在写一篇顶刊综述，我给你中文，帮我翻译成英文，要求AI率 5% 以下，逻辑要严谨，语句要通顺，要求口语化占比 30%，学术用于占 70%。
```

### 画图

> 以下内容提到的AI优先GPT，其次豆包。

#### 1、将文章内容复制给AI，拿到绘图建议。

参考提示词

```
请根据我的文章内容，结合顶刊顶会的插图思路，给出绘制图的建议，以下是我的文章内容：(粘贴文章内容)
```

这里可以补一句：给每张图写好caption，在图片插入论文时同时插入caption。

#### 2、将每一个建议单独发给AI，拿到绘图提示词。

参考提示词

```
(粘贴绘图建议)，请根据以上建议，生成AI绘图提示词，绘制符合顶刊顶会的插图，图中文字采用英文显示，以下为我的文章内容，确保绘图贴合文章，信息源于文中：(粘贴文章内容)
```

#### 3、将拿到的每一个提示词用绘图模式绘制出来，作为参考图。

#### 4、参考AI生成的图片，开始用PPT绘图，

- 图标可以从[阿里矢量图标库](https://www.iconfont.cn/)找。
- 也可以用原图的，将原图部分单独截出来，给豆包（提示词：变清晰），紧接着在清晰图像上方按钮，扣出主体，就得到了清晰可用的原图部分。

#### 文字部分手打即可。

#### 5、导出为PDF，上传overleaf，插入文中，插入caption。

### 表格

> 以下内容提到的AI优先GPT，其次豆包。

#### 1、将文章内容复制给AI，拿到绘图建议。

参考提示词

```
请根据我的文章内容，结合顶刊顶会的插表思路，给出绘制表的建议，以下是我的文章内容：(粘贴文章内容)
```

#### 2、从其他论文模板中找合适的模板，复制到文中，手动填充数据即可。
