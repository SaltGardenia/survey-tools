### 提示词

参考文献：

```
我要完成顶刊综述的部分写作任务，末尾会给你附上大纲，要求你根据大纲，给我梳理价值高的参考文献，参考文献占比：5年以上 10%，近五年 90%，近 3 年70%。直接给我文章链接，并给出二级标题下的写作思路。
```

翻译：

```
我在写一篇顶刊综述，我给你中文，帮我翻译成英文，要求AI率 5% 以下，逻辑要严谨，语句要通顺，要求口语化占比 30%，学术用于占 70%。
```
